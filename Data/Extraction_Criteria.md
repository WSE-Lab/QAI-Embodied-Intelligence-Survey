# Extraction Criteria

| RQ | Extraction Criteria |
| -- | ------------------- |
| RQ1 | The publication date of the primary study. <br> The publication venue of the primary study. <br> The venue type of the primary study, i.e., journal, conference. <br> The venue domain of the primary study, e.g., computer science, quantum. |
| RQ2 | The type of the quantum AI in each primary study, i.e., quantum, quantum-inspired, hybrid. <br> The classical base technique of the quantum AI in each primary study, e.g., reinforcement learning(RL), particle swarm optimization(PSO). <br> The number of agents in the embodied intelligence system in each primary study, i.e., single-agent, multi-agent. <br> The input modalities of embodied intelligence adopted in each primary study, e.g., proprioception, vision. |
| RQ3 | The integration method of quantum computing and classical AI techniques in each primary study: <br> &nbsp;&nbsp;&nbsp;&nbsp;1. Quantum-enhanced representation <br> &nbsp;&nbsp;&nbsp;&nbsp;2. Quantum-enhanced learning mechanism <br> &nbsp;&nbsp;&nbsp;&nbsp;3. Quantum algorithm integration |
| RQ4 | The exact components of the EI workflow that are augmented with QAI, i.e., perception, reasoning, execution. <br> The main functionality of the QAI, e.g., optimization, representation. <br> The classical AI approaches associated with QAI Across EI Modules, e.g., reasoning-RL. |
| RQ5 | The application context of the embodied intelligence in each primary study, e.g., generic robotics, manufacturing. <br> The capability and primary task of the embodied intelligence in each primary study, e.g., contorl, navigation path planning. <br> The mobility mode of the embodied intelligence in each primary study, e.g., fixed-base robot, wheeled robot. |
| RQ6 | The quantum-side execution of the quantum AI algorithm in each primary study, e.g., IBM cloud setting. <br> The quantum platform used for the quantum-side execution in each primary study, e.g., Qiskit, PennyLane. <br> The embodied-side platform used for the embodied-side execution in each primary study, e.g., Gazebo. |
| RQ7 | The availability of open-source repositories and implementation platforms used in the primary studies. |
